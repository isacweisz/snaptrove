# SnapTrove Deployment

This is a static website. Deploy the whole folder as the site root.

Recommended quick route:

1. Create a free Vercel, Netlify, or Cloudflare Pages project.
2. Upload this folder or connect it to a GitHub repository.
3. Set the custom domain to `getsnaptrove.com`.
4. In your domain registrar, point DNS to the hosting provider.

Pages:

- Homepage: `/`
- App demo: `/app.html`

Files required:

- `index.html`
- `app.html`
- `styles.css`
- `assets/snaptrove-logo.svg`
- `assets/travel-hero.jpeg`
