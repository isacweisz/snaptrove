Do guests need to pay?

No. Guests should upload, view, like, comment and react for free.

Are photos deleted after the event?

No. A Trove should feel like a permanent private memory vault. Event upload windows may end, but existing memories are not automatically deleted after 30 days.

What happens when storage is full?

New uploads pause and the owner sees an upgrade prompt. Existing photos remain safe, organised and ready to relive.

What is archive mode?

If a user stops paying, SnapTrove can archive the Trove instead of immediately deleting memories. Photos stay safe, uploads pause, AI processing and premium features are restricted, and the owner can reactivate by upgrading.

Can I remove photos?

Hosts should control what stays, what is shared and what can be printed. Owners can archive, delete, export, extend the upload window or upgrade storage.

Can schools and businesses use SnapTrove?

Yes. Business plans are designed for higher storage, admins, branding, permissions, exports and privacy controls.

Storage and retention

Your private memory vault, not a temporary upload folder.

Photos are not automatically deleted when the event ends. SnapTrove is designed so memories stay safe, organised and ready to relive, while storage is monetised through fair upgrades, Event Passes and Pro plans.

01

Stored properly

Original full-resolution files, optimised web versions and thumbnails live in cloud object storage, with metadata for users, Troves, permissions, comments, likes, reactions, albums, AI tags, faces and orders stored in the database.

02

No surprise deletion

If a free Trove reaches its storage limit, new uploads pause and the owner sees an upgrade prompt. Existing photos stay safe instead of disappearing.

03

Archive mode

If a plan ends, the Trove can be archived: photos stay safe, new uploads pause, AI processing and premium features are restricted, and the owner can reactivate by upgrading.
